# English Empire Institute - Web App & Management System 🇬🇧

A modern web application and internal management system built for the **English Empire Institute**. 

This project is currently in the early stages of development. The primary goal is to provide a high-conversion landing page for prospective students, a portal for families to manage enrollments, and a comprehensive dashboard for the institute's staff to handle academic and financial data.

### Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Mercado Pago Checkout

*More features and documentation will be added as the project evolves.*
